SWObjects - RDF Toolkit

Practical tools for manipulating RDF

SWObjects is a library and a set of tools for manipulating RDF data and queries.

Features

  • strict and generous query parsing
  • built-in webserver (in sparql executable)
  • direct mapping dump for MySQL, Postgres, Oracle
  • direct mapping query execution on MySQL, Postgres, Oracle
  • SPARQL Protocol support
  • SPARQL Graph Store Protocol support

Platforms

  • Linux 32 bit
  • Linux 64 bit
  • Windows
  • Mac OSX
  • perl
  • python
  • php
  • lua
  • java
